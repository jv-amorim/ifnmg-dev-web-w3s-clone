Tag HTML <tr>

Exemplo

Uma tabela HTML simples com três linhas, duas células de cabeçalho e quatro células de dados:

<table>
 <tr>
  <th>Mês</th>
  <th>Saldo</th>
 </tr>
 <tr>
  <td>Janeiro</td>
  <td>R$100,00</td>
 </tr>
 <tr>
  <td>Fevereiro</td>
  <td>R$80,00</td>
 </tr>
</table>

Definição e Uso

A tag <tr> define uma linha em uma tabela HTML.

Um elemento <tr> contém um ou mais elementos <th> ou <td>.
